Prerequisites

Install Yap prolog (tested on YAP 6.2.3), follow these instructions and using these options for configure:
../configure --enable-tabling=yes --enable-dynamic-loading

Install GSL library e.g. for Ubuntu:
sudo apt-get install libgsl0-dev libgsl0ldbl

Install boost library

Optional
In case of problems with BFL library, use these instructions to compile and install the patched BFL

Installation

Download the code:
svn checkout http://distributional-clauses.googlecode.com/svn/trunk/ distributional-clauses

Install/test library:
sh install.sh

Expected output (the number can be different)
testing sampling library...
yes
sample from gaussian(1,1): 1.42071748719105
Everything works!


Troubleshooting

In case of problems try to recompile, execute in random directory:
make clean && make

Test with
make test

otherwise try:
make clean && make sharedlocal

If the patched BFL has been compiled and installed manually execute instead:
make clean && make shared

In case of problems with BFL library (file liborocos-bfl.so/a), use these instructions to compile and install the patched BFL and perform:
make clean && make shared

Test again with
make test
